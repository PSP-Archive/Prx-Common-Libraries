cmlibCtrlHook_sample

# sample.prx(KernelModule)
1. A push on L_trigger + Triangle will repeal all buttons.
2. All buttons will be validated if L_trigger + Triangle is pushed once again.

# sample2.prx(UserModule)
1. A push on a Triangle button will swap Left / L_trigger & Right / R_trigger buttons.
2. It will return, if a Triangle button is pushed once again.
